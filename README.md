Once installed, you can use pyjamaz from your terminal or command line:

1. **Open your terminal or command line interface.**
2. **Navigate to the folder containing your images:** Use the `cd` command to change directories.
3. **Run the following command to optimize an image:**
   ```

   ```
   Replace `<your-image-file>` with the name of the image you want to optimize.

4. **Check the optimized image:** After running the command, the optimized image will replace the original, or you can specify an output file.
